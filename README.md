# Java Web 商城


参考[how2j.cn](http://how2j.cn/stage/39.html)


使用的技术:
* `Maven 项目`
* `Spring Boot`
* `Spring MVC`
* `Hibernate ORM`
* `Thymeleaf 模板引擎`
* `前后端分离`
* `RESTful API`
* `Shiro 安全验证`
* `Redis 缓存`
* `Nginx 动静分离`


## 实现功能
* 后台分类管理
* 后台用户管理
* 后台产品管理
* 后台产品图片管理
* 后台属性管理
* 后台产品属性值设置
* 后台订单管理
* 首页
* 注册
* 登录
* 退出
* 评论
* 产品详情页
* 分类页
* 搜索
* 购物车
* 订单流程
